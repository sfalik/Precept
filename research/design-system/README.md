# Design System Research

This folder is for design-system and UX research owned by the product-facing visual system.

Use it for evidence and synthesis that inform reusable visual and interaction rules, such as:

- semantic visual studies
- cross-surface UX research
- interaction-pattern analysis
- accessibility findings
- system-level visual language proposals

Do not use this folder for raw source captures, prototype files, or finalized specs.

- Source material belongs in `../references/`.
- Artifact critique belongs in `../reviews/`.
- Durable approved rules belong in `../foundations/` or other canonical system specs.
