Precept — Typography Exploration

Three approaches: distinctive geometric wordmark, pragmatic Inter, or no wordmark (icon only). Each shown at multiple sizes and in full context.

Approach A: Distinctive Geometric Wordmark

A custom-feeling typeface for "Precept" only. Body text / docs use a separate neutral face. This is most impactful when the logo needs to stand alone without an icon.

Approach C: No Wordmark (Icon Only)

How to evaluate

  • 48px — README hero area, landing page. Is there personality?
  • 24px — Section header, sidebar nav. Still distinctive?
  • 14px — NuGet listing, VS Code marketplace. Readable?
  • Paired with code — The wordmark sits next to .precept code often. Does it clash or complement?
  • Maintenance cost — Distinctive wordmark = more design surface. Inter = zero friction. Icon-only = icon must be excellent.
