How to evaluate

  • Badge visibility — which stands out in a NuGet search results page (mostly white/light bg)?
  • Syntax keyword color — which feels most natural as the DSL keyword color you'd see daily?
  • State diagram — which accent makes the transition arrows most readable?
  • Icon at 32px — which is still visible and distinctive at VS Code sidebar scale?
  • .NET ecosystem fit — teal, amber, and indigo each break from .NET's purple in different ways.
