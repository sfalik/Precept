1. Positioning LOCKED

Primary frame

Category creation — "domain integrity engine"

No competitor occupies this frame. Precept defines the category, then becomes synonymous with it. Same pattern as Temporal ("durable execution"), Docker ("containerization"), Terraform ("infrastructure as code").

Secondary frame

AI-native — "the contract AI can reason about"

Deterministic semantics + structured tool APIs make Precept uniquely suited for AI authoring and validation. This is the "why now" signal, not the primary identity.

Single-sentence positioning

"Precept is a domain integrity engine for .NET — a single declarative contract that binds state, data, and business rules into an engine where invalid states are structurally impossible."

2. Narrative Archetype LOCKED

Category Creator: "This is a new kind of thing."

Like Temporal's "durable execution" or Tailwind's "utility-first CSS." Requires category education but earns the strongest position.

README narrative arc

  1. One sentence: what it is (resolution)
  2. One sentence: why it's different (implicit conflict)
  3. One code sample (proof)
  4. Feature list or badge row (comprehensiveness signal)
The name "Precept" carries narrative weight — it means "a strict rule of action." The name IS the story.

4. Color System LOCKED

17 shades in 6 families. Background #0c0c0f. Every color has a fixed meaning across all surfaces. Nothing is decorative.

Semantic color as brand

Most tools pick colors for aesthetics, then assign meanings later. Precept's palette is the reverse — every color exists because the compiler knows something about the code and needs a way to say it. The system is organized along two dimensions:

  • Dimension 1 — what kind of thing: States (nouns — where you are), Events (verbs — what happens), Data (variables — what you know). Each gets its own hue family so a line of code is instantly scannable without reading it.
  • Dimension 2 — runtime evaluation: Verdicts (enabled / blocked / warning) are reserved for runtime outcomes. They never appear in syntax highlighting, so green and red can't be confused with authoring-time colors.
  • Rules sit at the intersection — they aren't a category of thing (like states or events). They produce verdicts about categories. Gold is the only warm hue in a cool-toned palette, so constraint syntax interrupts visually: when you see gold, you know it's a rule.
  • Structure is scaffolding — intentionally quiet. Declaration, action, and glue keywords fade behind the semantic content they frame.

Structure · Indigo · 239–245°

Scaffolding — intentionally sub-AA contrast, fades behind the content it frames. Three tiers create hierarchy: declarations define the schema, actions drive the machine, glue is connective tissue.

6. Visual Language LOCKED

Primary surface — DSL syntax as hero

The .precept file format IS the brand. The code is the hero image.

precept LoanApplication field ApplicantName as string nullable field RequestedAmount as number default 0 invariant RequestedAmount >= 0 because "Amount cannot be negative" state Draft initial state UnderReview state Approved state Declined event Submit with Applicant as string on Submit assert Applicant != "" because "Name required" from Draft on Submit -> set ApplicantName = Submit.Applicant -> transition UnderReview from UnderReview on Approve -> transition Approved from UnderReview on Decline -> transition Declined

Secondary surface — state diagrams

The VS Code preview panel renders diagrams using the same semantic palette. Code and diagram share visual language.
