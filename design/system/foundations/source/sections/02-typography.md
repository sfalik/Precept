### Type is doing semantic work, not brand decoration.

Typography in Precept is a three-way split between governed identity, reading voice, and pressure marking. The section only works if those jobs stay narrow.

- **Geist Mono** handles governed language, labels, chips, code, and navigation.
- **Geist** handles sustained reading and explanatory prose.
- *Italic* is reserved for constraint pressure inside an already-known construct family.

### The rule is discipline, not variety.

The cue only stays trustworthy if the system does not spend it casually. If mono governs, sans explains, and italic means constrained attention, the reader can trust the document at speed.
