### Semantic color should answer “what kind of thing is this?” before anything else.

The five authored families stay fixed: Structure, State, Event, Data, and Rule. Runtime verdicts and comments are real signals, but they are not additional authored construct families.

- Indigo is the structural lane.
- Violet is state identity.
- Cyan is event direction.
- Slate is governed data.
- Gold is human-readable rule pressure.

### Verdicts are overlays, not new semantics.

Green, red, and yellow report run-time outcome. They do not replace authored identity. The system only stays legible if the authored lane remains visible underneath the runtime pressure.
