Why this stays an island

The gallery is not just repeated prose. It is a specific scanning structure with paired specimens, chips, tables, and verdict cards. Authoring that in raw markdown would either collapse the layout or push too much presentational syntax into the prose source.

Authoring split

Markdown still owns the section frame, tier explanation, and proof notes. The island only owns the dense gallery body.
