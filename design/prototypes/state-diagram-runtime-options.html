Visual cues

        

          
Colour
Violet marks state names and state nodes across authored and rendered views.

          
Typography
Open
Mono PascalCase keeps state names readable as named lifecycle coordinates.

          
Form
Closed
Rounded 8px state node is the baseline, then dot, fill, dimmed hatch or dashed edge, and double border layer on top.

        

      

      

        
Semantic modifiers

        

          
Initial
Open
Origin dot marks the initial state.

          
Terminal
Closed
Double border marks a terminal state.

          
Constrained
Closed
Italic label marks a constrained state.

          
Unreachable from initial
Closed
Heavy dimming and interior hatch mark a state unreachable from the initial state.

          
Unreachable from current
Closed
Heavy dimming and dashed edge mark a state unreachable from the current state.

          
Current
Open
Filled node marks the current state.
