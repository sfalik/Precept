Semantic modifiers

        

          
transition
Close→Closed
Directional pairing reads the trigger and destination as one move.

          
self-transition
Recheck↻Open
Loop treatment reads the trigger while the lifecycle coordinate stays the same.

          
no-transition
Close∅
Empty destination marker reads the event as resolved without a state change.

          
rejected
Closerejected
Rejection lands on the trigger as a run-time verdict, not a second event family.

          
constraint failure
