# Research

Technical research that informs Precept's language design and capability roadmap.

This folder accumulates over time — findings here feed into design decisions in `docs/` and feature proposals reviewed by the team.

## Structure

| Folder | Owner | Purpose |
|--------|-------|---------|
| `dsl-expressiveness/` | George + Steinbrenner | Analysis of how comparable DSLs and libraries express concepts that Precept currently requires more statements to express. Feeds feature proposals. |
| `language-references/` | George | PLT concepts, type system patterns, binding/evaluation models — theoretical grounding for DSL design decisions. |

## Related

- `brand/references/` — Brand-adjacent research (hero snippet benchmarks, README conventions, positioning). Owned by J. Peterman.
- `docs/PreceptLanguageDesign.md` — The DSL spec that research informs.
